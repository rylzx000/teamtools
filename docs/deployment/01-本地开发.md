# 本地开发

## 环境前提

- Windows 本机开发。
- 已安装 Python、Node.js、pnpm、Git。
- 不依赖 Docker。

## 本地目录

```text
D:\project\teamtools
```

运行数据目录在工程内：

```text
D:\project\teamtools\data
```

## 初始化数据目录

```powershell
D:\project\teamtools\scripts\init-data.ps1
```

作用：创建 `data/` 下的配置、模块资料和任务目录，并创建工程根目录下的 `logs/`。

## 本地启动

启动 API：

```powershell
D:\project\teamtools\scripts\dev-api.ps1
```

构建前端：

```powershell
D:\project\teamtools\scripts\build-frontend.ps1
```

访问：

```text
http://127.0.0.1:8000
```

## 数据目录结构

```text
data/
  teamtools.db
  config/
    app.yaml
    modules/
      fpa/
        module.yaml
        systems.yaml
  modules/
    fpa/

      examples/
        input/
        expected/
  tasks/
    fpa/
      {task_id}/
        input/
        ai/

```
