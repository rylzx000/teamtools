# FPA 工作量评估模块

## 模块定位

FPA 模块是平台首批落地的业务模块，用于把用户提交的需求文本或 Markdown 文件转换为功能点法工作量评估结果。

首版目标是跑通完整后台流程：

- 用户提交 FPA 评估任务。
- 后端生成 AI 请求包。
- 前端调用外部模型并回传结构化结果。
- 后台脚本校验结构化数据。
- 后台脚本基于 Excel 模板生成评估文件。
- 用户查看任务状态并下载 Excel。
